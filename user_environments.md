# User Environments

## Resources

These are available to each user:

- **RAM:** 2 GB
- **CPU:** 1
- **Storage Limits:** 10G per user (configurable for special purposes)

## Kernels

The base image includes both Python and R kernels.

## Development Environments

- JupyterLab
- Jupyter Notebook
- VS Code
- RStudio
- Shiny
